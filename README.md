# WPML automatic language with GeoIP

Automatically changes the load of language in WPML based on the GeoIP service

# Features

1. Compatible with WP Engine's GeoIP service
1. Uses MaxMind GeoIP service

# Requirements

- php: >=5.3.1
- WordPress
